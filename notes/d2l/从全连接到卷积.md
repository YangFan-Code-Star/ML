# 6.1 从全连接到卷积

## 一句话总结

**大白话**：全连接层处理图像时，每个像素都连到每个神经元，参数多到爆炸。但图像有两条常识：一只猫不管出现在左上还是右下都是猫（平移不变），而且判断「这里是不是边缘」只看附近几个像素就够了（局部性）。顺着这两条砍掉冗余连接、让同一个小窗口在整张图上滑动复用，就得到了[[卷积]]。

**严谨说法**：卷积层由两条对图像合理的先验导出——**平移不变性**（同一特征到处用同一检测器）与**局部性**（只看邻域）；它本质是参数共享 + 稀疏连接的特殊全连接层，大幅减少参数。

## 本节解决的问题

- 全连接处理图像为何参数爆炸？
- 平移不变性与局部性如何约束出卷积结构？
- 通道、特征图的含义？

## 核心概念 / 公式 / API

### 两条先验 → 卷积
- **平移不变性**：检测器与位置无关 → 权重在空间上共享。
- **局部性**：输出只依赖输入的局部窗口 → 卷积核很小。
- 由此把全连接的巨大权重张量约化为小卷积核（互相关运算）：

$$H_{i,j}=\sum_{a,b} V_{a,b}\, X_{i+a,\,j+b}$$

| 概念 | 含义 |
| --- | --- |
| 卷积核 / 滤波器 | 一组共享权重的小窗口 |
| 特征图 | 卷积输出的空间响应 |
| 通道 | 多个滤波器或输入的颜色/特征维 |
| 感受野 | 输出某点对应的输入区域 |

## 易错点

- 把深度学习「卷积」与数学卷积混淆——实现是**互相关**（不翻转核）。
- 误以为卷积是全新结构，其实是受约束的全连接。

## 复习卡片

- Q: 卷积来自哪两条先验？
  A: 平移不变性与局部性。
- Q: 卷积相对全连接的优势？
  A: 参数共享 + 局部连接，参数量与计算大幅减少。
- Q: 深度学习的「卷积」实际是什么运算？
  A: 互相关（不翻转卷积核）。
