# 11.6 动量法 (Momentum)

## 一句话总结

**大白话**：把下山的「人」换成「带惯性的球」。球会累积之前的方向：一直往下的方向越滚越快，左右乱晃的方向正负抵消、晃动变小。于是在又窄又陡的峡谷里不再来回撞墙，收敛更快更稳。

**严谨说法**：动量法用历史[[梯度]]的指数加权平均作为更新方向，像「带惯性的小球」滚下损失面，抑制震荡、加速在病态/狭长峡谷方向上的收敛。

## 本节解决的问题

- 病态（条件数大）时 SGD 为何来回震荡？
- 动量如何平滑更新方向？
- 动量系数 β 的含义？

## 核心概念 / 公式 / API

$$\mathbf{v}_t=\beta\mathbf{v}_{t-1}+\mathbf{g}_t,\qquad \mathbf{w}_t=\mathbf{w}_{t-1}-\eta\mathbf{v}_t$$

- $\mathbf{v}_t$ 是梯度的指数加权移动平均（EWMA）。
- β（如 0.9）越大惯性越强，等效平均最近 $1/(1-\beta)$ 步梯度。
- 在一致方向上累积加速，在震荡方向上相互抵消。

| API | 说明 |
| --- | --- |
| `torch.optim.SGD(params, lr, momentum=0.9)` | 带动量 SGD |

## 最小可运行代码

```python
import torch

opt = torch.optim.SGD(net.parameters(), lr=0.01, momentum=0.9)
# 手写版
v = [torch.zeros_like(p) for p in params]
def momentum_step(params, v, lr, beta):
    with torch.no_grad():
        for p, vi in zip(params, v):
            vi[:] = beta * vi + p.grad
            p -= lr * vi
            p.grad.zero_()
```

## 易错点

- β 过大（如 0.99）会过冲，需配合更小学习率。
- 动量与学习率耦合：等效步长约放大 $1/(1-\beta)$ 倍。
- 初期动量未充满（偏差），少数实现会做偏差修正。

## 复习卡片

- Q: 动量的更新方向是什么？
  A: 历史梯度的指数加权移动平均。
- Q: β=0.9 大致等效平均多少步梯度？
  A: 约 1/(1-0.9)=10 步。
- Q: 动量解决了什么问题？
  A: 病态/峡谷地形下 SGD 的来回震荡。
