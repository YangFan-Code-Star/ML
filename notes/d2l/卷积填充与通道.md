# 6.2-6.4 图像卷积、填充与步幅、多输入多输出通道

## 一句话总结

**大白话**：[[卷积]]核就是个小窗口在图上滑动算加权和。填充（padding）= 在边上补一圈零，免得越滑越小、边角信息丢失；步幅（stride）= 一次跨几格，跨得大图就缩得快（下采样）。每个输出通道是一个独立的滤波器；1×1 卷积不看空间、只在通道之间做混合（升降维）。

**严谨说法**：卷积层用互相关计算特征图；填充 (padding) 控制输出尺寸、步幅 (stride) 控制下采样；多输入通道按通道求和，多输出通道即多个独立滤波器，1×1 卷积用于通道融合。

## 本节解决的问题

- 卷积输出尺寸如何计算？
- padding/stride 各起什么作用？
- 多通道卷积的权重形状与运算规则？

## 核心概念 / 公式 / API

### 输出尺寸公式
输入 $n_h\times n_w$，核 $k_h\times k_w$，填充 $p$，步幅 $s$：

$$\left\lfloor\frac{n_h-k_h+p_h+s_h}{s_h}\right\rfloor \times \left\lfloor\frac{n_w-k_w+p_w+s_w}{s_w}\right\rfloor$$

- 常用 `padding = (k-1)/2`（奇数核）保持尺寸不变。

### 多通道
- 多输入通道 $c_i$：每通道一个核，结果逐通道相加 → 单张特征图。
- 多输出通道 $c_o$：权重形状 $(c_o, c_i, k_h, k_w)$。
- **1×1 卷积**：不看空间，只在通道维做线性组合（等价逐像素全连接）。

| API | 说明 |
| --- | --- |
| `nn.Conv2d(in_c, out_c, kernel_size, padding, stride)` | 二维卷积层 |

## 输入输出形状

- 输入 `(B, c_i, H, W)` → 输出 `(B, c_o, H', W')`，`H'/W'` 由上式决定。

## 最小可运行代码

```python
import torch
from torch import nn

conv = nn.Conv2d(3, 16, kernel_size=3, padding=1, stride=1)
X = torch.randn(2, 3, 32, 32)
print(conv(X).shape)        # (2,16,32,32)  padding=1 保持尺寸

down = nn.Conv2d(16, 32, kernel_size=3, padding=1, stride=2)
print(down(conv(X)).shape)  # (2,32,16,16)  stride=2 下采样
```

## 易错点

- 算输出尺寸时漏掉步幅/填充导致维度对不上。
- 多通道权重形状记错：是 `(out, in, kh, kw)`。
- 误以为 1×1 卷积无用——它是跨通道信息融合与降维的利器。

- 下采样的另一手段是池化 [[汇聚层]]。

## 复习卡片

- Q: 保持输出尺寸的常用填充？
  A: 奇数核取 `padding=(k-1)/2`。
- Q: 多输出通道卷积权重形状？
  A: `(c_o, c_i, k_h, k_w)`。
- Q: 1×1 卷积的作用？
  A: 跨通道线性组合（融合/升降维），不改变空间尺寸。
