# 3.5 图像分类数据集 (Fashion-MNIST)

## 一句话总结

**大白话**：Fashion-MNIST 就是「升级版手写数字」——10 类衣服鞋子的小灰图（28×28），全书拿它当统一练习题。本节学会怎么把它下载下来、转成张量、再切成一小批一小批喂给模型。

**严谨说法**：Fashion-MNIST 是 10 类、28×28 灰度的服饰图像数据集，作为全书分类实验的标准基准；本节学会用 `torchvision` 加载、用 `DataLoader` 批量迭代并做预处理。

## 本节解决的问题

- 如何下载/加载标准图像数据集？
- 如何用 `transforms` 做张量化与归一化？
- 如何构建可多进程读取的小批量迭代器？

## 核心概念 / 公式 / API

| API | 作用 |
| --- | --- |
| `torchvision.datasets.FashionMNIST` | 下载/加载数据集 |
| `transforms.ToTensor()` | PIL → `(C,H,W)` 浮点张量，缩放到 [0,1] |
| `data.DataLoader(..., shuffle, num_workers)` | 批量、打乱、多进程读取 |

- 训练集 60000 张，测试集 10000 张；标签 0-9 对应 T 恤/裤子/…/短靴。

## 输入输出形状

- 单图 `(1, 28, 28)`；一个 batch `(B, 1, 28, 28)`，标签 `(B,)`。
- 送入全连接前需 `nn.Flatten()` → `(B, 784)`。

## 最小可运行代码

仓库脚本：`d2l/code/3.5图像分类数据集.py`、`d2l/code/3.5图像分类数据集_本地版.py`

```python
import torch
from torch.utils import data
from torchvision import datasets, transforms

trans = transforms.ToTensor()
train = datasets.FashionMNIST("./data", train=True, transform=trans, download=True)
test = datasets.FashionMNIST("./data", train=False, transform=trans, download=True)

def load(batch_size=256):
    return (data.DataLoader(train, batch_size, shuffle=True, num_workers=4),
            data.DataLoader(test, batch_size, shuffle=False, num_workers=4))

train_iter, test_iter = load()
X, y = next(iter(train_iter))
print(X.shape, y.shape)        # (256,1,28,28) (256,)
```

## 易错点

- Windows 上 `num_workers>0` 需放在 `if __name__ == "__main__":` 保护下，否则多进程报错。
- 忘记 `ToTensor` 归一化，输入像素仍是 0-255，训练不稳。
- 本地数据放在 `data/FashionMNIST/raw/`，离线时设 `download=False`。

## 复习卡片

- Q: Fashion-MNIST 的图像尺寸与类别数？
  A: 28×28 灰度，10 类。
- Q: 全连接前为什么要 Flatten？
  A: 把 `(B,1,28,28)` 展平成 `(B,784)` 适配 `nn.Linear`。
- Q: Windows 多进程读取注意事项？
  A: 入口加 `if __name__ == "__main__":` 保护。
