# 8.1 序列模型

## 一句话总结

**大白话**：股价、文字、音乐这类数据有先后顺序，前后还互相影响（不像独立的图片）。预测未来要靠过去——但「过去」可能无限长，于是要么只看最近几步（马尔可夫假设：当前主要由最近 τ 步决定），要么用一个不断更新的「记忆」（[[隐状态]]）把全部历史浓缩起来。

**严谨说法**：序列数据的样本不独立、有时间顺序；用自回归（基于过去预测未来）建模时，常用**马尔可夫假设**把无限历史截断为固定长度 $\tau$，或用[[隐状态]]概括全部历史。

## 本节解决的问题

- 序列数据为何不能套用 i.i.d. 假设？
- 自回归模型与隐变量自回归的区别？
- 马尔可夫假设如何让条件分布可计算？

## 核心概念 / 公式 / API

| 概念 | 含义 |
| --- | --- |
| 自回归 | 用过去若干步预测当前：$P(x_t\mid x_{t-1},\dots,x_{t-\tau})$ |
| 隐变量自回归 | 维护隐状态 $h_t$ 概括历史（RNN 思想） |
| 马尔可夫假设 | 当前只依赖最近 $\tau$ 步 |
| 序列分解 | $P(x_1,\dots,x_T)=\prod_t P(x_t\mid x_{<t})$ |

- 预测越往后误差累积越大（多步预测退化）。

## 最小可运行代码

```python
import torch
from torch import nn

# 用前 tau 步预测下一步（马尔可夫）
tau = 4
features = torch.stack([x[i:i + tau] for i in range(len(x) - tau)])
labels = x[tau:].reshape(-1, 1)
net = nn.Sequential(nn.Linear(tau, 10), nn.ReLU(), nn.Linear(10, 1))
```

## 易错点

- 用未来信息预测过去（数据泄露）——必须严格按时间切分。
- 单步预测好≠多步预测好，注意误差累积。

## 复习卡片

- Q: 马尔可夫假设的作用？
  A: 把当前预测的依赖截断为最近 τ 步，使分布可计算。
- Q: 隐变量自回归与普通自回归的区别？
  A: 前者用隐状态概括全部历史，后者直接用固定窗口。
- Q: 多步预测为何变差？
  A: 误差逐步累积放大。
