# 参数初始化 (Parameter Initialization)

> **大白话**：训练开始前给权重随机「定个初始大小」。定得太大，信号一层层放大到爆炸；太小，一层层缩小到消失；还不能全设成一样（否则所有神经元学成同一个）。要不大不小、互不相同。

## 目标
让每层**前向输出**与**反向梯度**的方差跨层大致保持不变，避免[[梯度消失与爆炸]]。

## 常用方案
- **Xavier/Glorot**：$\mathrm{Var}(w)=\dfrac{2}{n_{in}+n_{out}}$，适配 tanh。
- **Kaiming/He**：$\mathrm{Var}(w)=\dfrac{2}{n_{in}}$，适配 ReLU。

```python
import torch.nn as nn
def init_weights(m):
    if isinstance(m, nn.Linear):
        nn.init.xavier_uniform_(m.weight)
        nn.init.zeros_(m.bias)
net.apply(init_weights)
```

## 易错点
- 全 0 / 全相同初始化 → 对称性无法打破，神经元学到相同特征。
