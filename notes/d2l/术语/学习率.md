# 学习率 (Learning Rate)

> **大白话**：下山时「每一步迈多大」。迈太大容易冲过头甚至摔下去（发散），迈太小走得龟速（收敛慢）。它是训练里最关键、最需要调的旋钮。

## 定义
参数更新公式 $\theta \leftarrow \theta - \eta\,\nabla_\theta L$ 中的步长系数 $\eta$。

## 关键事实
- 过大 → 震荡/发散；过小 → 收敛慢、易陷停滞；
- 与 batch size、动量系数耦合，需联动调；
- **自适应**：AdaGrad/RMSProp/Adam 为每个参数自动调步长 → [[Adam]]、[[AdaGrad与RMSProp]];
- **调度**：训练中动态变化（warmup + 衰减）→ [[学习率调度器]]。
