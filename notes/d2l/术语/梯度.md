# 梯度 (Gradient)

> **大白话**：站在山坡上，梯度就是「哪个方向最陡上坡」的箭头，箭头长度代表坡有多陡。想下山（减小损失）就朝它的反方向走。

## 定义
多元函数 $f$ 对各输入变量偏导组成的向量 $\nabla f = (\partial f/\partial x_1, \ldots, \partial f/\partial x_n)$，指向函数值增长最快的方向。

## 在训练中的角色
- 优化沿**负梯度**方向更新参数：$\theta \leftarrow \theta - \eta\,\nabla_\theta L$。
- 由**链式法则**经反向传播自动计算（[[微积分与自动微分]]）。

## 相关陷阱
- **梯度消失/爆炸**：深网络里梯度连乘后趋于 0 或炸开 → [[数值稳定性与初始化]]。
- PyTorch 梯度默认累加，每个 batch 前要 `zero_grad()`。
