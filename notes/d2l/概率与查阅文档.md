# 2.6-2.7 概率与查阅文档

## 一句话总结

**大白话**：概率是「描述不确定性」的语言。条件概率 = 已知一件事后另一件事的可能性；贝叶斯 = 用新证据更新旧判断；期望 = 平均会得到多少；方差 = 波动有多大。这套语言是分类损失（交叉熵）和「极大似然」的根基。顺带学会不会的 API 怎么自查。

**严谨说法**：概率论提供处理不确定性的语言：随机变量、[[条件概率]]、[[贝叶斯定理]]、期望与方差，是分类损失（交叉熵）与[[极大似然估计]]的理论基础；并学会用 `dir`/`help` 查 PyTorch 文档。

## 核心概念 / 公式 / API

| 概念 | 公式 | 说明 |
| --- | --- | --- |
| 条件概率 | $P(A\mid B)=\dfrac{P(A,B)}{P(B)}$ | 已知 B 下 A 的概率 |
| 贝叶斯定理 | $P(A\mid B)=\dfrac{P(B\mid A)P(A)}{P(B)}$ | 后验 ∝ 似然 × 先验 |
| 边缘化 | $P(A)=\sum_B P(A,B)$ | 求和消去变量 |
| 期望 | $E[X]=\sum_x xP(x)$ | 分布的中心 |
| 方差 | $\mathrm{Var}[X]=E[(X-E[X])^2]$ | 分布的离散程度 |

### 采样
```python
import torch
from torch.distributions import multinomial

fair = torch.ones(6) / 6
counts = multinomial.Multinomial(1000, fair).sample()
print(counts / 1000)        # 频率随采样数趋近 1/6（大数定律）
```

### 查阅文档
```python
import torch
print(dir(torch.distributions))   # 列出模块内对象
help(torch.ones)                  # 查看函数文档/签名
```

## 最小可运行代码

仓库脚本：`d2l/code/2.6概率.py`

```python
import torch
from torch.distributions import multinomial

probs = torch.tensor([0.1, 0.2, 0.7])
samples = multinomial.Multinomial(10000, probs).sample()
print(samples / samples.sum())    # ≈ [0.1, 0.2, 0.7]
```

## 易错点

- 混淆 $P(A\mid B)$ 与 $P(B\mid A)$（贝叶斯就是连接二者）。
- 采样次数太少时频率偏离真实概率，误以为分布不对。

## 复习卡片

- Q: 贝叶斯定理公式？
  A: $P(A\mid B)=P(B\mid A)P(A)/P(B)$。
- Q: 大数定律的直观含义？
  A: 采样越多，频率越接近真实概率。
- Q: 不知道某函数用法怎么办？
  A: 用 `dir()` 列对象、`help()` 看文档。
