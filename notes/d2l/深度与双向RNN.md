# 9.3-9.4 深度循环神经网络与双向 RNN

## 一句话总结

**大白话**：深度 RNN = 把循环层叠几层，下层的笔记给上层接着读，表达力更强。双向 RNN = 一遍从左往右读、一遍从右往左读，再把两份理解拼起来，这样每个位置都同时看到「前文+后文」。但正因为它偷看了「后文」，所以不能用来做「猜下一个词」的生成任务（那时后文还不存在）。

**严谨说法**：深度 RNN 把多层循环层堆叠以增强表达力；双向 RNN 同时从左到右和从右到左处理序列，融合上下文，适合「已知整句」的特征编码任务，但不能用于自回归生成。

## 核心概念 / 公式 / API

| 结构 | 说明 |
| --- | --- |
| 深度 RNN | 多层堆叠，第 l 层隐状态作为第 l+1 层输入；`num_layers>1` |
| 双向 RNN | 前向 + 后向两套隐状态拼接，输出维度翻倍 |

- 双向需要完整序列（看得到未来），适合编码/标注；**不能用于语言模型生成**（会泄露未来）。

| API | 说明 |
| --- | --- |
| `nn.LSTM(..., num_layers=2)` | 深层 |
| `nn.LSTM(..., bidirectional=True)` | 双向，输出 hidden×2 |

## 输入输出形状

- 双向输出最后一维为 `2*hidden`；隐状态第一维为 `2*num_layers`。

## 最小可运行代码

```python
import torch
from torch import nn

birnn = nn.LSTM(28, 256, num_layers=2, bidirectional=True)
X = torch.randn(35, 32, 28)
Y, _ = birnn(X)
print(Y.shape)        # (35, 32, 512)  双向 -> 256*2
```

## 易错点

- 用双向 RNN 做自回归生成 → 泄露未来信息，预测无效。
- 双向输出维度翻倍，下游全连接输入维度要相应调整。
- 层数过深而不加正则/残差易过拟合或难训。

## 复习卡片

- Q: 双向 RNN 为何不能做语言模型生成？
  A: 它依赖未来信息，自回归生成时不可得。
- Q: 双向 RNN 输出维度如何变化？
  A: 拼接前后向，变为 `2*hidden`。
- Q: 深度 RNN 如何堆叠？
  A: 下层隐状态作为上层输入，`num_layers>1`。
