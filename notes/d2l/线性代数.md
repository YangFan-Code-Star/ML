# 2.3 线性代数

## 一句话总结

**大白话**：标量是一个数、向量是一排数、矩阵是一张表、张量是更高维的一摞表。神经网络说到底就是在不停做「矩阵乘法」（一层变换）和「算长度」（范数，用来度量大小/误差）——这些就是它的数学骨架。

**严谨说法**：标量、向量、矩阵、[[张量]]构成数据的多维表示；点积、矩阵-向量积、矩阵乘法与各类[[范数]]是神经网络前向计算和损失度量的数学骨架。

## 本节解决的问题

- 不同阶张量如何表示与运算？
- 沿指定轴求和/均值、保持维度 (`keepdim`) 怎么用？
- 点积、矩阵乘法、范数在网络中分别对应什么？

## 核心概念 / 公式 / API

| 概念 | 公式 | API | 用途 |
| --- | --- | --- | --- |
| 点积 | $\mathbf{x}^\top \mathbf{y} = \sum_i x_i y_i$ | `torch.dot` | 加权和 |
| 矩阵-向量积 | $\mathbf{A}\mathbf{x}$ | `torch.mv` | 一层线性变换 |
| 矩阵乘法 | $\mathbf{A}\mathbf{B}$ | `torch.mm` / `@` | 批量线性变换 |
| L2 范数 | $\lVert \mathbf{x} \rVert_2 = \sqrt{\sum_i x_i^2}$ | `torch.norm` | 权重衰退 |
| L1 范数 | $\sum_i \lvert x_i \rvert$ | `x.abs().sum()` | 稀疏正则 |
| F 范数 | 矩阵元素平方和开方 | `torch.norm` | 矩阵大小度量 |

### 降维求和
```python
A.sum(dim=0)               # 沿轴0求和，维度减少
A.sum(dim=1, keepdim=True) # 保持维度，便于广播
A.cumsum(dim=0)            # 累加不降维
```

## 输入输出形状

- `A(m,n) @ B(n,p) → (m,p)`；`A(m,n) @ x(n,) → (m,)`。
- `sum(dim=i)` 去掉第 `i` 维；`keepdim=True` 保留为 1。

## 最小可运行代码

仓库脚本：`d2l/code/2.3线性代数.py`

```python
import torch

A = torch.arange(20.).reshape(5, 4)
print(A.T.shape)                       # 转置 -> (4, 5)
print(A.sum(dim=0).shape)              # 沿行求和 -> (4,)
x = torch.arange(4.)
print(torch.dot(x, x))                 # 点积
print((A @ x).shape)                   # 矩阵-向量积 -> (5,)
print(torch.norm(torch.ones(4, 9)))    # F 范数 = 6.0
```

## 易错点

- `sum(dim=1)` 与 `sum(dim=0)` 方向相反，先想清楚要消除哪一维。
- 广播除法做归一化时常需 `keepdim=True`，否则形状不匹配。
- `*` 是逐元素乘 (Hadamard)，不是矩阵乘；矩阵乘用 `@`/`mm`。

## 和前后章节 / 真实项目的连接

- 矩阵乘法即全连接层核心；范数即 [[权重衰退]] 的正则项。
- 为 [[微积分与自动微分]] 的梯度运算打基础。

## 复习卡片

- Q: L2 范数公式？
  A: $\sqrt{\sum_i x_i^2}$。
- Q: `keepdim=True` 的作用？
  A: 归约后保留被消维度为 1，便于后续广播。
- Q: `*` 和 `@` 区别？
  A: `*` 逐元素乘，`@` 矩阵乘。
