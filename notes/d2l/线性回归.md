# 3.1-3.3 线性回归（从零 + 简洁）

## 一句话总结

**大白话**：给每个特征配一个权重、再加个偏置，把它们加权求和当预测值——这就是线性回归（本质是给特征「打分相加」）。预测得准不准用「差的平方」来罚，再用梯度下降一点点拧权重。它是「模型+损失+优化+训练循环」这套四件套最小、最完整的样板。

**严谨说法**：线性回归用 $\hat{y}=\mathbf{x}^\top\mathbf{w}+b$ 拟合连续目标，配平方损失与小批量随机梯度下降；它是理解「模型 + 损失 + 优化 + 训练循环」四件套的最小完整范例。

## 本节解决的问题

- 如何用线性模型 + 平方损失刻画回归任务？
- 训练循环（前向 → 算损失 → 反向 → 更新）怎么写？
- 从零实现与 `nn` 简洁实现如何一一对应？

## 核心概念 / 公式 / API

| 组件 | 公式 / API | 说明 |
| --- | --- | --- |
| 模型 | $\hat{y}=\mathbf{X}\mathbf{w}+b$ | `nn.Linear(d, 1)` |
| 损失 | $\frac{1}{2}(\hat{y}-y)^2$ | `nn.MSELoss()` |
| 优化 | $\mathbf{w}\leftarrow\mathbf{w}-\eta\nabla$ | `torch.optim.SGD` |
| 数据 | 小批量迭代 | `data.DataLoader` |

- 平方损失来自高斯噪声假设下的[[极大似然估计|极大似然]]。
- 解析解存在（最小二乘），但深度学习统一用 SGD 迭代。

> **大白话**：为什么用「差的平方」而不是「差的绝对值」？因为假设真实值=预测值+高斯噪声时，让数据最可能出现（极大似然）算出来的，恰好就是最小化平方误差。

## 输入输出形状

- `X(n, d) @ w(d, 1) + b → (n, 1)`；标签 `y` reshape 为 `(n, 1)`。

## 最小可运行代码

仓库脚本：`d2l/code/3.2线性回归从零开始实现 copy.py`、`d2l/code/3.3线性回归简洁实现.py`

从零实现核心：
```python
import torch

w = torch.normal(0, 0.01, (2, 1), requires_grad=True)
b = torch.zeros(1, requires_grad=True)

def net(X): return X @ w + b
def loss(yhat, y): return (yhat - y.reshape(yhat.shape)) ** 2 / 2

lr = 0.03
for epoch in range(3):
    for X, y in data_iter(batch_size, features, labels):
        l = loss(net(X), y).sum()
        l.backward()
        with torch.no_grad():
            for p in (w, b):
                p -= lr * p.grad / batch_size
                p.grad.zero_()
```

简洁实现：
```python
from torch import nn
from torch.utils import data

dataset = data.TensorDataset(features, labels)
loader = data.DataLoader(dataset, batch_size=10, shuffle=True)
net = nn.Sequential(nn.Linear(2, 1))
loss = nn.MSELoss()
trainer = torch.optim.SGD(net.parameters(), lr=0.03)
for epoch in range(3):
    for X, y in loader:
        trainer.zero_grad()
        loss(net(X), y).backward()
        trainer.step()
```

## 易错点

- 从零版忘记 `with torch.no_grad()` 更新参数 → 把更新算入计算图。
- 标签形状 `(n,)` 与预测 `(n,1)` 不一致触发广播，损失算错。
- 学习率过大发散、过小收敛慢。

## 复习卡片

- Q: 平方损失的概率来源？
  A: 高斯噪声假设下的极大似然估计。
- Q: 训练循环四步？
  A: 前向预测 → 计算损失 → 反向求梯度 → 更新参数（并清零梯度）。
- Q: `nn.Linear(2,1)` 内部等价于？
  A: $X@w+b$。
