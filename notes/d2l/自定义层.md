# 5.4 自定义层 (Custom Layers)

## 一句话总结

**大白话**：自己造一个层和造一个块没区别，都是继承 `nn.Module`。不带权重的层（比如做个减均值）只写 `forward`；带权重的层要把权重用 `nn.Parameter` 包一下，这样优化器才「看得见」、才会去更新它。

**严谨说法**：自定义层和自定义块本质相同——继承 `nn.Module`；无参数层只需 `forward`，带参数层用 `nn.Parameter` 声明可学习权重以便被 autograd 与优化器追踪。

## 本节解决的问题

- 如何写一个不带参数的工具层（如中心化）？
- 如何写带可学习参数的自定义全连接层？
- `nn.Parameter` 与普通张量区别？

## 核心概念 / 公式 / API

| 概念 | 说明 |
| --- | --- |
| 无参层 | 只实现 `forward`，无 `nn.Parameter` |
| 有参层 | 用 `nn.Parameter(tensor)` 声明，自动 `requires_grad` 并注册 |
| `nn.Parameter` | Module 的特殊属性，会被 `parameters()` 收集 |

## 最小可运行代码

```python
import torch
from torch import nn
import torch.nn.functional as F

class CenteredLayer(nn.Module):           # 无参数层
    def forward(self, X):
        return X - X.mean()

class MyLinear(nn.Module):                # 带参数层
    def __init__(self, in_units, units):
        super().__init__()
        self.weight = nn.Parameter(torch.randn(in_units, units))
        self.bias = nn.Parameter(torch.zeros(units))
    def forward(self, X):
        return F.relu(X @ self.weight + self.bias)

layer = MyLinear(5, 3)
print(layer(torch.rand(2, 5)).shape)      # (2, 3)
print([n for n, _ in layer.named_parameters()])  # ['weight', 'bias']
```

## 易错点

- 用普通 `torch.Tensor` 当权重 → 不会被 `parameters()` 收集，优化器更新不到。
- 忘了 `nn.Parameter` 包裹导致梯度不更新。
- 自定义层的输入形状假设要与上游一致，注意维度对齐。

## 复习卡片

- Q: 可学习参数必须用什么声明？
  A: `nn.Parameter`。
- Q: 无参数层最少要实现什么？
  A: 只需 `forward`。
- Q: 为什么普通张量当权重不被更新？
  A: 不被 `parameters()` 收集，优化器看不到。
