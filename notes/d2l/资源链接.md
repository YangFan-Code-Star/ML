## 工具与安装（PyTorch 路线）

```bash
# d2l 工具包
pip install d2l

# PyTorch（按本机 CUDA 选择对应命令，见 pytorch.org）
pip install torch torchvision
```

- PyTorch 官网（获取正确安装命令）: https://pytorch.org/get-started/locally/
