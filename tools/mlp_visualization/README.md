# MLP 可视化工具

这里存放这次整理的神经网络可视化脚本。

## 脚本

- `4.4_MLP曲线拟合可视化.py`：训练一个 1D MLP，展示拟合过程、第二隐藏层激活、最终拟合曲线。
- `函数复合可视化.py`：展示一般函数复合 `f(x)` 和 `g(f(x))`。
- `sigmoid函数复合可视化.py`：展示 `sigmoid(x)` 和 `sigmoid(sigmoid(x))`。
- `relu函数复合可视化.py`：展示 `ReLU(x)` 和 `ReLU(ReLU(x))`。
- `relu网络表达能力可视化.py`：展示 `Linear -> ReLU -> Linear -> ReLU` 如何增加表达能力。
- `linear加权相加过程可视化.py`：展示隐藏层特征如何被 `Linear` 加权相加形成输出。
